UCSC 2022-2023 Academic Major Calculator

A program for UCSC Advisors/Students to calculate what 2022-2023 majors students qualify for given the completed class history.

How to Use the Program:

> Search Up/Scroll through the list of all UCSC courses
> Select courses that you have previously completed AND passed with a satisfactory grade
> Once the courses have been chosen, click on "Calculate Qualified Majors"

Qualified Majors:

> Creates a list of all majors you can consider. Each major will have a class list following the title to show what courses in the list you chose lined up with the major. All majors that are considered "qualified" for you are only considering prerequisite courses needed to declare the major.
